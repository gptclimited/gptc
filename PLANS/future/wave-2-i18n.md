# Wave 2 — Multi-Language Support (Draft)

**Status:** Backlog  
**Depends on:** Wave 1 CMS recommended, client language priorities

## Goal

Serve English and French audiences (minimum); additional languages TBD.

## Technical approach

- `next-intl` or Next.js locale routing (`/en`, `/fr`)
- Translated content in CMS with `locale` field
- `hreflang` alternate links in metadata
- Language switcher in site header
- Typography verified for all scripts

## Acceptance criteria (draft)

- [ ] Home, About, Contact, Services hub in EN + FR
- [ ] hreflang validated in Search Console
- [ ] Language preference persists across navigation

## Estimated effort

8–12 weeks (includes translation workflow + top pages)
