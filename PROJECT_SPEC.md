# Global Training Network Website PRD

## Project Overview

Create a modern, professional, SEO-optimized website for Global Training Network (GTN), a consulting, training, mentorship, and educational organization serving:

* Domestic Students
* International Students
* Newcomers
* Families
* Educational Institutions
* Community Organizations
* Faith-Based Organizations

The website should establish credibility, communicate services clearly, generate consultation inquiries, and rank well on search engines.

Primary CTA:

* Book a Consultation
* Request Information
* Contact Us

Secondary CTA:

* Apply for Educational Consulting
* Join Training Programs
* Partner With Us
